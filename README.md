# Polyshop - Tregu Dixhital i Shqipërisë

Polyshop është një marketplace modern shqiptar i ndërtuar me React dhe Tailwind CSS. Platforma ofron një përvojë të luksit për të blerë dhe shitë produkte dhe shërbime me ndjekjen e lëvizjeve të çmimeve në kohë reale.

## Karakteristikat

✨ **Ndëraface Premium** - Dizajn i frymëzuar nga Polymarket me pamje fintech

🌙 **Dark Mode** - Mbështetje të plotë për modalitetin e errët

📱 **Responsive Design** - Përvojë e përsosur në të gjithë pajisjet

⚡ **Performancë** - Ngarkesë e shpejtë dhe animacione të ëmta

🎨 **Framer Motion** - Animacione të avancuara dhe tranzicione

🇦🇱 **100% Albanian** - Të gjitha interfacat në gjuhën shqipe

## Teknologjitë e Përdorura

- **React 18** - Biblioteka e React për ndërtimin e ndërfaqes
- **Tailwind CSS** - Framework CSS utility-first
- **Framer Motion** - Biblioteka për animacione
- **Lucide React** - Ikonat moderne

## Instalimi

```bash
npm install
npm start
```
